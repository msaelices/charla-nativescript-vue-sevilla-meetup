NativeScript

“Open source framework for building truly native mobile apps with Angular, Vue.js, TypeScript, or JavaScript”
  • Creado por Telerik en 2015
  • Adquirido por Progress
  • Runtime JS que dirige el UI
  • Acceso con JS al 100% APIs nativas iOS y Android

NativeScript vs Ionic

  • No hay WebView
  • Mayor rendimiento
  • Menor tiempo de start-up

NativeScript vs React Native

Muy similar. Diferencias sutiles

  • No ligado a framework JS específico
  • Acceso total a librerías nativas

Webapp con Vue

            
<html>
  <body>
    <div id="app"></div>
    <script src="js/bundle.js"></script>
  </body>
</html>
            
          
            
import Vue from 'vue'
import App from './App.vue'

new Vue({
  render: h => h(App),
}).$mount('#app')

Mobile App con NS-Vue

              
import Vue from 'nativescript-vue';
import App from './components/App.vue';

new Vue({
  render: h => h('frame', [h(App)])
}).$start();

Creación de App

  • Seguir documentación en nativescript-vue.org
  • Se presupone NativeScript ya instalado
            
$ npm install -g @vue/cli @vue/cli-init
$ vue init nativescript-vue/vue-cli-template my-app
$ cd my-app
$ npm install
$ tns run android --bundle
$ # or
$ tns run ios --bundle
